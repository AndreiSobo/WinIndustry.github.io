Week 1: Basics

Published on October 6, 2022

This week I began to look at the basics of HTML and how to use them.

My previous knowledge of HTML, CSS, and webpage creation is fairly limited, so I decided to spend some time looking at the way this style of code works. I did some basic research about the use of tags and the way HTML is different from code I had used before, as you cannot simply press run and check its functionality.

  1. I set up a webpage with the base code provided, and the basic layout style.
  2. I then created 3 different .html files for each part of the website so far, the learning journal, the tutorial, and the contact.
  3. Using link tags, I ensured that there was a link to each of these pages, and altered them slightly so I could check this worked.
  4. Finally, I learnt how to upload the start of my website to the server, 'brighton.domains'. Once I understood how this worked, the process was not too difficult. I learnt from this point that organisation would be a key part of this project and without it, it would be difficult to track the process of the webpage.
